步骤 1️⃣: 推送到 GitHub
  git add .
  git commit -m "Initial commit: AHR999 Index"
  git push

步骤 2️⃣: 连接 Cloudflare Pages
  1. 访问 https://dash.cloudflare.com
  2. Workers & Pages → Create → Connect Git
  3. 选择仓库，点击部署

步骤 3️⃣: 配置 KV 和 Cron
  1. 创建 KV 命名空间: AHR999_DATA
  2. 绑定到项目（变量名: AHR999_DATA）
  3. 添加 Cron Trigger: 0 * * * *
  4. 访问 /init 初始化数据

详细步骤请查看: 快速开始.md
